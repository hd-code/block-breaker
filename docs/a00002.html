Detailed Description

Encapsulates the enumeration of depth test policies.

The standard policy of the depth test is lesser, i.e. if the depth value of a new drawn pixel is lesser than the depth of the pixel already registered inside of the depth buffer, then the last one is replaced by the new pixel. However, there are other useful policies, e.g. switching off the depth test at all while rendering GUI elements.

enum gfx::SDepthTest::ETest
Enumerator
Lesser 

The pixel is drawn if its depth value is lesser than the depth value of the pixel already in the depth buffer.

Equal 

The pixel is drawn if its depth value is equal to the depth value of the pixel already in the depth buffer.

Off 

The pixel is always drawn, i.e. the depth test is deactivated.
